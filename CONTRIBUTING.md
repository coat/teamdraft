Every one is welcome to contribute. This was obviously vibe-coded, so LLM-generated code is acceptable, but let's try and keep communication with each other using our own voices and prose - particularly in PRs/issues.

## Guidelines

### Linting

We use standardrb - ensure you run `standardrb --fix` before committing.

### Testing

Follow [Four-Phase Test](https://thoughtbot.com/blog/four-phase-test) pattern when setting up tests. It's OK to repeat yourself in tests - it's better to have readable tests than DRY ones.

[Let's not](https://thoughtbot.com/blog/lets-not) use let.
